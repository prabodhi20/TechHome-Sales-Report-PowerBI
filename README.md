# TechHome Sales Report – Power BI Dashboard

## Overview
An interactive Power BI dashboard designed to analyze TechHome’s sales, profit, and discount impact across categories, segments, and regions. 
This project demonstrates industry-standard BI practices, including KPI cards, conditional formatting, and storytelling elements.

## Key Features
- **KPIs**: Total Sales, Total Profit, Profit Margin %, Orders, Profit Status
- **Conditional Formatting**: Profit/Loss cards auto-color green/red
- **Consistent Palette**: Navy, teal, green/red for clarity

## Data Source
Sample Superstore dataset (Sales, Orders, Discount, Region, Segment, Category, Sub-Category).

## Business Insights
- Technology category leads profit growth.
- Discounts have a strong negative impact on margins.
- Standard Class shipping dominates.
- Consumer segment drives half of total sales.
- Furniture and Bookcases show losses; need corrective pricing or cost control.

## Tools & Skills
- Power BI Desktop
- DAX (measures, dynamic title, conditional formatting)
- Data modeling (Date table, relationships)
- Visualization design & storytelling

## Outcome
Delivered a client-ready dashboard enabling management to track profitability, identify loss-making categories, and optimize discount strategies.
